Process Scheduler Simulator (FCFS)
Description
This project is a C program that simulates the First Come First Serv
Tasks are executed in the order they arrive. Once a task starts, it ru
calculates Finishes Time, Total Time, and Wait Time for each task a
wait times.
Features
- FCFS scheduling
- User input for number of tasks, arrival time, and duration time
- Calculates finishes time, total time, and wait time
- Displays results in a clear table
- Calculates average total time and average wait time
Technologies Used
Programming Language: C
Concepts: Arrays, loops, input/output
How to Run
1. Compile: gcc fcfs.c -o fcfs
2. Run: ./fcfs
Output Format
Task | Arrival | Duration | Finishes | Total Time | Wait Time
